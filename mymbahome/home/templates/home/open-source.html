O que significa ser open source?

A filosofia de desenvolvimento open source diz respeito ao acesso do código fonte de um programa. Um programa que contenha o código fonte aberto e disponível e que possa ser acessado, modificado e estudado de forma gratuita pode ser considerado um software open source.

Para saber mais, clique aqui e leia sobre este conceito. Ou clique aqui para saber mais sobre as categorias de código fonte aberto.

Enquanto o Mymba (que ainda está crescendo aos poucos) não descreve a licensa dos softwares desenvolvidos para o projeto, entre em contato através das redes sociais ou do formulário do site para solicitar mais Informações sobre a licensa dos programas (:

Mymba Edu

O impacto social que buscamos não se restringe apenas aos animais, mas também ao homo sapiens. O Mymba Edu é um programa educacional que está intimamente ligado ao própósito pelo qual os projetos de tecnologia desenvolvidos são open-source. Seu objetivo é transferir conhecimento sobre programação, inovação e empreendedorismo aos indivíduos ligados ao projeto.

O programa educacional com foco em programação aproveita da característica de código aberto dos sistemas para integrar os participantes ao desenvolvimento tecnológico dos softwares do projeto.

Tecnologias por trás do Mymba

Para construir o ecossistema ideal a fim de atingir os objetivos institucionais, diverssa ferramentas são utilizadas.

  • Arquitetura de serviços distribuídos (em migração para microsserviços)
  • Docker e docker compose
  • HTML5 + CSS3 + Bootstrap
  • Arduino + Raspberry Pi
  • Banco de dados MySQL e Mongodb
  • Linguagem de programação Python e JavaScript
  • E outras que ainda serão utilizadas para o desenvolvimento de novas features
